# Kinboni E-Commerce Platform
Kinboni is a high-performance, enterprise-grade full-stack e-commerce platform designed to deliver a seamless shopping experience. Built with a modern tech stack, it features a robust multi-role management system and optimized media handling to ensure scalability and speed.


## 🚀 Features
- Multi-Role Access Control (RBAC): Features a secure 5-role management system (Super Admin, Admin, Manager, Seller, Customer) for streamlined operations.
- Optimized Media Delivery: Integrated Cloudinary API for dynamic image transformation and high-performance media storage.
- Responsive UI: Built a seamless, mobile-first shopping experience using responsive Tailwind CSS and Shadcn UI components.
- Scalable Backend: Engineered with Next.js, Node.js, and Express.js, leveraging Prisma ORM and PostgreSQL for robust data management.
### Dynamic Homepage
- **High-Impact Hero Sliders:** Interactive and visually engaging hero sliders designed to showcase featured promotions and high-priority store content.
- **Weekly Offer Banners:** Dedicated banner sections to highlight weekly deals, boosting user engagement and promotional visibility.
- **Best Sellers Countdown:** Dynamic "Best Sellers" section integrated with real-time countdown timers to create urgency and drive customer conversions.

### Advanced Filtering
- **Price Range**: Users can set custom price constraints using an interactive slider or manual numerical input.
- **Multi-level Categories:** Robust navigation system supporting tiered hierarchies such as Fruits, Meats, and Dairy for simplified product discovery.
- **Product Attributes:** Granular filtering options allowing users to refine searches based on Color, Brand, and real-time Stock Status.

### User Interface
- **Responsive Design:** Fully optimized, mobile-first design ensuring a seamless browsing experience across all desktop and mobile devices.
- **Smart Interaction:** Includes integrated indicators for wishlist and cart management, providing immediate feedback for a superior shopping experience.
- **Product Interaction:** Features clean, modular product cards equipped with "Quick View" and "Add to Cart" functionalities for efficient user interaction.

## 🛠️ Tech Stack

- **Frontend:** Built with React.js and Next.js for a high-performance, server-side rendered application.
- **Styling:** Styled using Tailwind CSS for a modern, responsive design and Styled Components for modular UI components.
- **Deployment:** Deployed on Vercel to ensure fast global content delivery and high availability.

### Installation Steps for Haatzo Project

### Step 1: Clone the Repository
Copy the repository to your local machine:
```bash
git clone https://github.com/mdnishanchowdhury/kinboni-fullstack.git
```
This will create a folder named Haatzo in your current directory containing the project files.

### Step 2: Navigate to the Project Folder
```bash
cd kinboni-fullstack
```
Now you are inside the project folder where the package.json file is located.

### Step 3: Install Dependencies
```bash
pnpm install
```

### Step 4: Run the Development Server
```bash
pnpmdev
```
This will start the project locally. By default, it runs on http://localhost:3000.

### Step 5: Open in Browser
Open your browser and visit:
http://localhost:3000
You should now see the Haatzo website running on your local machine.
